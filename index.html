Welcome to TaskTrack

This tutorial will show you how to manage your tasks using the system.

Step 1

Click OPEN on the home page to access the task menu.

Step 2

Select ADD TASK to enter new tasks.

Step 3

Select UPDATE TASK to edit or delete tasks.

Step 4

Select VIEW TASK to monitor completed and pending tasks.

Step 5

Click SAVE TASKS to export your tasks as a document.

End of Tutorial

You can now start organizing your academic tasks.
